# terminal-browser

A simple terminal-based browser written in Python.  
This project maded around 7 mins so do not expect good stuff.

## Requirements
- Python 3.x

## Installation
```
git clone https://github.com/halocxent/terminal-browser.git
cd terminal-browser
```

## Running
```
py web.py
```
or
```
python3 web.py
```
